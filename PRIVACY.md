# Privacy

DeadTime v0.1.0-beta runs locally in your browser or desktop WebView preview.

## Data Storage

DeadTime stores app state in local browser storage on your machine:

- countdown timer state
- resume point text
- background wait labels
- checklist items
- scratchpad notes
- theme, compact mode, sound, and hint settings

No data is sent to a server by the app.

## Network Use

The local preview server only serves files from this project folder to `127.0.0.1`.

## Metadata

The included logo PNG has been stripped down to required PNG image chunks only. No author name, local path, editor history, prompt text, or token data is intentionally stored in the project files.

Git commits use the GitHub noreply email configured for the publishing account.
